# DISHA AI CALCULATOR

DISHA AI CALCULATOR is a smart calculator that solves a wide range of mathematical problems using natural language input.  
Users can type questions in plain English, and the system returns accurate results using AI-based reasoning.


## Features
- Natural language based calculations
- Single calculator for all math problems
- Supports arithmetic, algebra, percentages, and word problems
- Clean and simple user interface
- Backend-based architecture (secure)

##  Problem Types Supported
- Basic arithmetic ( + − × ÷ )
- Percentages and discounts
- Powers and roots
- Algebraic equations
- Word problems
- Averages, ratios, proportions
- Simple interest and compound interest
- Geometry (area, perimeter, volume)

##  Example Inputs
What is 25% of 400?
Solve 2x + 5 = 15
Square root of 1024
I bought 3 items at 299 each with 10% discount. What is the total?

## Tech Stack
- HTML, CSS, JavaScript
- Node.js
- Express.js
- AI-based language model
